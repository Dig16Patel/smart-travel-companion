# Backend - Smart Travel App

This directory will contain the backend server for the Smart Travel App.

## Technology Stack
- Node.js
- Express
- Firebase Admin SDK

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up your `.env` file with necessary API keys and Firebase credentials.
3. Run the development server:
   ```bash
   npm run dev
   ```
   *(Or run `npm start` to run normally without nodemon)*
