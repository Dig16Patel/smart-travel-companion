# Smart Travel App

A comprehensive travel guide application with multimodal transport options.

## Project Structure

- **frontend/**: The Flutter mobile application.
- **backend/**: The backend server API (Setup pending).

## Getting Started

### Frontend (Flutter App)

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    flutter pub get
    ```
3.  Run the app:
    ```bash
    flutter run
    ```

### Backend (Node.js/Express)

1.  Navigate to the `backend` directory:
    ```bash
    cd backend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up environment variables (create a `.env` file and configure Firebase, etc.).
4.  Run the server in development mode:
    ```bash
    npm run dev
    ```
    *(Or use `npm start` for standard run)*
